Herramientas de trabajo

Mobirise

Usamos Mobirise por la versatilidad y variedad de instrumentos que nos ofrece, para poder diseñar una pagina web comprensible y estetica.

Github

Github nos ofrece un servicio de WebHosting completamente gratis y permite el acceso de todo el equipo de trabajo, para cambiar o mejorar la informacion que recopilamos
